## âœ¨ Features

- **Classic gameplay:** Play the standard two-player Tic-Tac-Toe game.
- **Time travel (move history):** View the full move history and jump to
  previous moves.
- **Move locations:** Each move shows its board coordinates in the format (row,
  col).
- **Winning highlight:** Highlights the three squares that caused a win and
  shows a draw message when no one wins.
- **Sort moves:** Toggle the move list order between ascending and descending.
- **Status display:** Clear game status showing next player, winner, or draw.
